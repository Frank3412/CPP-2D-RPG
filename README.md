# CPP-2D-RPG

A 2D RPG game built from scratch in Modern C++ using SDL3.

## Current Features

- SDL3 rendering
- Tile-based world
- Camera that follows the player
- Delta-time movement
- Optimized visible tile rendering
- Collision detection with wall sliding
- Object-oriented Player class
