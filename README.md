# CPP-2D-RPG

A 2D RPG game built from scratch in Modern C++ using SDL3.

## Features

- SDL3 window creation
- Renderer
- Keyboard input
- Player movement
- Sprite rendering
- CMake build system
- vcpkg dependency management

## Technologies

- C++20
- SDL3
- SDL3_image
- CMake
- Git
- GitHub
